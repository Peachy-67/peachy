Goal:
Build a web app that detects red flags in conversations and helps users identify manipulation, gaslighting, and harmful behavior.

MVP Features

1. Conversation Analyzer
User pastes conversation
AI analyzes for red flags
Display verdict and badges

2. Result Visualization
Verdict component
Flag badges
Confidence score

3. Shareable Results
Screenshot friendly layout
Share button
Copy result

4. Alert System
High-risk behaviors trigger alert
User notified immediately

5. Dashboard
Real-time conversation monitoring
Live updates
Alert stream
