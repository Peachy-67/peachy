# Decision Quality Runbook

Repeatable checklist for high-quality execution decisions.

## Pre-action checklist
- Confirm objective and desired outcome in one sentence.
- Confirm scope boundaries and what is out-of-scope.
- Identify whether action is reversible or irreversible.
- If major/irreversible, ask user approval before proceeding.
- Break work into the smallest useful step.
- Define how success will be verified before running the step.

## Uncertainty checklist
- List unknowns that could change the decision.
- State current confidence level (tentative / confirmed / strong).
- Note assumptions explicitly.
- Surface top risks early.
- Choose a low-risk validation step to reduce uncertainty.
- If uncertainty remains high, pause and ask for clarification.

## Promotion checklist
- Verify this insight/procedure appears in 2+ independent sources or has explicit user confirmation.
- Confirm entry is durable (not a one-off tip or temporary context).
- Confirm duplicates are merged rather than added.
- Record promotion signal used.
- Record confidence level.
- Record concise source/reason note.

## Execution checklist
- Execute one small step at a time.
- Verify output after each step.
- Log actions, outcomes, and errors in run logs.
- Adjust plan based on evidence, not assumptions.
- Move task state via execution loop: INBOX → ACTIVE → DONE/WAITING.
- End with: outcome, errors, and next action.
